# instantloantoday.org

Static landing pages (English + French) for the Canada personal loan offer.

## Pages

- `index.html` — English landing (same content as `engch1.html`)
- `engch1.html` — English
- `fren2.html` — Français

## Assets

- `gus.png` — hero image (copied from project template)

## Deploy

Upload files to your host for **www.instantloantoday.org** or configure GitHub Pages / Cloudflare Pages on this repository.

## GitHub

```bash
git init
git add .
git commit -m "Initial commit: landing pages"
git branch -M main
git remote add origin https://github.com/YOUR_USER/instantloantoday.org.git
git push -u origin main
```
